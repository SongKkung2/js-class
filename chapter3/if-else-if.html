if~else if

두가지 이상의 조건식과 정해 놓은 조건을 만족하지 않았을때 실행되는 코드로 이뤄져있습니다.

            if (조건식1){
                조건식 1이 참이면 실행되는 코드
            } else if(조건식2){
                조건식 2가 참이면 실행되는 코드
            } else if (조건식~){
                조건식~~~
            } else {
                모두 일치하는 것이 없으면, 
            }
