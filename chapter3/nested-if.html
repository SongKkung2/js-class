중첩 if 예시

질의 응답창을 이용해 로그인 하는 프로그램을 작성할것

미리 정해진 아이디와 패스워드, 사용자가 입력한 아이디 패스워드를 비교해서 아이디가 일치하면 패스워드를 비교해서 그것마저 일치했을때 로그인이 성공하고, 아이디가 일치하나 패스워드를 비교해서 일치하지 않다면 로그인 실패. 또는 아이디 마저 미리 정해진 아이디와 다르다면, 그냥 실패!
