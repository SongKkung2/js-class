제어문

프로그램의 실행 순서, 흐름을 제어하는 문장 (문법, 구문)이다.

제어문은 크게 1. 조건문 , 2. 선택문 , 3. 반복문으로 구분한다.

조건문

조건식의 값이 true,false 에 따라 각각 다른 코드를 실행하는 문법

특정 조건에 의해서 프로그램 실행이 분기 (=나누어) 지는 것.

  1. 단순 if
  2. if ~ else
  3. if ~ else ~ if
  4. 중첩 if

단순 if

조건식이 참일때만 실행되는 구문, 거짓일때는 실행하지 않음

            if(조건식) {
                ....자바스크립트 코드(들)....;
            }

if~else

단순 if 는 참일떄만 실행되지만, if~ else 는 참일떄 거짓일때 실행하는 구문이 각각 존재함

            if(조건식){
                true 일때 실행코드
            } else {
                false 일때 실행코드
            }
